# Simple RESTful API

a simple RESTful API demonstration, including `register`, `login`, and `menu` endpoints

| HTTP Method | Endpoint        |
| ----------- | --------------- |
| GET         | `/api/menu`     |
| GET         | `/api/menu/:id` |
| POST        | `/api/menu`     |
| POST        | `/api/register` |
| POST        | `/api/login`    |
| PUT         | `/api/menu/:id` |
| DELETE      | `/api/menu/:id` |

# How to Install

```bash
npm install # to install all dependencies
node app.js # running the program
```
